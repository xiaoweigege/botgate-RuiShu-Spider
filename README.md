# 前言

本次采集的目标站点是`江苏省农村产权交易信息平台`，网址`http://www.jsnc.gov.cn/jygg/tzgg/index.html`。没想到这么小的网站都上瑞数了，爬虫真是越来越难了。回归正题，该网站的防御机制是在`cookie`上做了手脚，`cookie`名称为`YwnBCHQI8xgWI5a`。
